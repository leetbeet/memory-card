# Memory Card Game

A browser-based memory card game built with **React**, inspired by _Plants vs Zombies_, where players try to click all unique cards without repeating any.

## Features

- Click or **keyboard-navigate** unique cards to increase your **score**
- **Shuffles cards** after each click
- Tracks your **best score**
- Shows a **"You win!" dialog** when all cards are clicked correctly
- **Fully keyboard-accessible** with `Tab`, `Enter`, and `Space` support
- Responsive layout for **desktop and mobile screens**
- Accessible UI with clear visual feedback

## Technologies Used

- **React** (functional components, hooks: `useState`)
- **JavaScript (ES6 Modules)** for game logic and state management
- **CSS3** for styling, including Flexbox and CSS Grid
- **Vite** or React DOM for fast development and bundling
